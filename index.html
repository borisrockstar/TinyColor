TinyColor

→ A fast, small JavaScript color manipulation library

TinyColor is meant for inputting colors and outputting them as, or comparing them with, different colors. Input is meant to be as permissive as possible. The following color types are supported:

  • RGB
  • Hex
  • HSL
  • HSV
  • Name (from HTML color codes)

Usage

	

		TinyColor is still in active development, but the basic syntax is:

<script type='text/javascript' src='tinycolor.js'></script>



<script type='text/javascript'>
var tiny = tinycolor("green");
alert(tiny.toHexCSS());
// alerts #ffa500
</script>
